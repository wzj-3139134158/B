登录

1. 招生管理系统
接口功能

登陆

HTTP请求方式

get

URL

http://127.0.0.1:7001/login

支持格式

json

请求参数说明
名称 类型 必填 说明
name string 是 登录的账号
password string 是 登陆的密码
方法

get

接口描述

分为管理员登陆和普通用户登陆 管理员有特定的账号 所有普通账号均为管理员创建 管理员可以实现增删改查功能 普通用户仅能查看招生信息
